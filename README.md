# 📚 诗词赏析 · 中小学必背 302 首

一个纯静态的诗词赏析网站，精选 **中小学必背诗词 302 首**，每首配有原文、译文、注释、创作背景和深度赏析。

## ✨ 功能

- 🏠 **首页** — 每日一诗、热门推荐、全部分类入口
- 📜 **详情页** — 完整展示诗词 + 逐句朗读 + 名句高亮 + 相关推荐
- 📚 **分类列表** — 按唐诗/宋词/元曲/现代诗/先秦/五代/明清分类浏览
- 🔍 **全文搜索** — 可按标题、作者、原文、名句搜索
- 👤 **作者专题** — 聚合每位诗人的全部作品
- 🔊 **语音朗读** — 支持语速调节和音色切换（默认/女声/男声）
- 🌓 **夜间模式** — 一键切换明暗主题
- 📱 **响应式** — 手机和电脑均可流畅浏览

## 🚀 快速启动

### 方式一：直接打开（不推荐）

双击 `index.html` — 但浏览器安全策略会拦截本地 JSON 加载，导致搜索和分类功能不可用。

### 方式二：本地 HTTP 服务（推荐）

```bash
# 需要有 Node.js
npx http-server . -p 8080 -c-1 --cors
```

访问 http://127.0.0.1:8080

### 方式三：在线部署

推送到 GitHub 后，开启 GitHub Pages 即可：

```
https://你的用户名.github.io/PoetryCollection/
```
